# nuxt3-intelligence

`nuxt3` 智能工具集合

<br />

## Usage

### install

```shell
npm i nuxt3-intelligence
```

### Program

```ts
import { getStaticDepsFromNuxtConfig } from 'nuxt3-intelligence'

const deps = getStaticDepsFromNuxtConfig()

deps // Get all static deps for modules and extends
```
